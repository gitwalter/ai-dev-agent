```yaml
  integration_work: ["integrate", "connect", "api", "service", "external"]
  user_facing_changes: ["ui", "dashboard", "interface", "visualization"]
```

#### **üéØ Context-Aware Intelligence**
```bash

# Automatic context detection:
"Create React dashboard component"        ‚Üí Detects CODING + UI ‚Üí Auto-creates story
"Write unit tests for auth module"       ‚Üí Detects TESTING ‚Üí No story (part of other work)
```

- **Perfect Traceability**: All significant work automatically tracked
